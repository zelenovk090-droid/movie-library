# Movie Library (Личная кинотека)

**Автор:** [Ваше Имя Фамилия]

## Описание

Movie Library — это десктопное GUI-приложение для ведения личной библиотеки фильмов. Данные хранятся в JSON.

## Функциональность

- Добавление фильмов (название, жанр, год, рейтинг)
- Таблица с фильмами
- Фильтрация по жанру и году
- Удаление фильмов
- Сохранение в JSON
- Проверка ввода (год — число, рейтинг 0-10)

## Запуск

python main.py
